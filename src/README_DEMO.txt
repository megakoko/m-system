Демонстрационная версия модульной медицинской информационной системы M-System

*****************************************************************************
*****************************************************************************

Запуск программы осуществляется с помощью исполняемого файла loader.

После запуска необходимо в настройках программы выбрать СУБД SQLite и 
задать имя базы данных (например, postgres).

Для того, чтобы начать работу с системой, необходимо нажать на кнопку "Вход в 
систему", затем ввести логин "admin" и пароль "pw" без кавычек, после чего 
в главном окне система покажет список загруженных модулей. Данные для входа в 
систему могут быть изменены в модуле "Пациенты".
